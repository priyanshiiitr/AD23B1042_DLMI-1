This repository contains my assignments for the Deep Learning for Medical Imaging course, covering key medical image analysis tasks such as retinal vessel extraction, brain MRI tumor segmentation, and cell nuclei separation. The implementations use classical image processing and deep learning techniques including thresholding, watershed segmentation, and mask-based analysis. Each notebook includes dataset handling, preprocessing, segmentation, evaluation, and visualization. These assignments demonstrate practical applications of computer vision and deep learning in medical imaging.
